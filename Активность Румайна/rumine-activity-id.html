Таблица активности форума Румайна 2011-2019 по сообщениям

В этой таблице показано количество отправленных пользователями постов на форуме Румайна за каждый месяц с момента его основания.

Особенности

  • Активность считается по всему форуму, не только ФЧ
  • Метод подсчета – id сообщений.
  • Единица измерения – количество сообщений.
  • На количество постов сильно влияет спам сообщений через расширенный ответ (актуально для 2013).
  • Для 2011-начала 2012 годов возможны небольшие неточности в определении статистики, потому что тем того времени практически не осталось.

Опции

  • Отображение страниц: вместо количества сообщений в месяц будет отображаться количество страниц

  • Отображение "в день": вместо общего количества сообщений / страниц в месяц будет отображаться среднее количество в день
