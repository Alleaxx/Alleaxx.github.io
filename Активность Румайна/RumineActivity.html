Таблица активности ФЧ Румайна 2013-2019 по страницам

В этой таблице указана вся известная активность по годам существования ФЧ. Статистику по 2013 году можно считать весьма примерной, так как во время подсчёта несколько ФЧ уже было удалено. В остальном она вполне соответствует действительности, учитывая, конечно, общую методику подсчета.

Особенности

  • Таблица показывает активность исключительно форумного чата и не затрагивает остальной форум. При этом многие её данные могут не соответствовать текущему состоянию форума из-за удаления старых ФЧ, некоторых отдельных постов и прочих изменений.
  • Используется метод подсчета по страницам. Это значит, что мы сначала узнаём номер страницы с 1-м числом месяца, а затем номер страницы с 1-м числом следующего месяца, и разницу делим на количество дней в месяце. необходимое значение.
  • Единица измерения "активности" – страницы в день. Так что если у месяца значение активности 10, то следует считать что каждый день этого месяца в среднем писалось по 200 сообщений.
  • Все данные этой страницы перенесены из моей внутренней таблицы на компьютере, что порождает беспорядок синхронизации из-за моей лени, невнимательности и прочего-прочего. Но и так сойдет, как грица.

Опции

  • Отображение цветов: каждая ячейка имеет свой фон в зависимости от показателя активности

Таблица активности форума Румайна 2011-2019 по сообщениям

В этой таблице показано количество отправленных пользователями постов на форуме Румайна за каждый месяц с момента его основания.

Особенности

  • Активность считается по всему форуму, не только ФЧ
  • Метод подсчета – id сообщений.
  • Единица измерения – количество сообщений.
  • На количество постов сильно влияет спам сообщений через расширенный ответ (актуально для 2013).
  • Для 2011-начала 2012 годов возможны небольшие неточности в определении статистики, потому что тем того времени практически не осталось.

Опции

  • Отображение страниц: вместо количества сообщений в месяц будет отображаться количество страниц

  • Отображение "в день": вместо общего количества сообщений / страниц в месяц будет отображаться среднее количество в день
